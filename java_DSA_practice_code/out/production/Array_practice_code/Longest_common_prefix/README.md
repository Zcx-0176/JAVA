# 最长公共前缀 (Longest Common Prefix)

## 题目描述

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 `""`。

---

## 示例

### 示例 1

**输入：** `strs = ["flower","flow","flight"]`  
**输出：** `"fl"`

### 示例 2

**输入：** `strs = ["dog","racecar","car"]`  
**输出：** `""`  
**解释：** 输入不存在公共前缀。

---

## 提示

- `1 <= strs.length <= 200`
- `0 <= strs[i].length <= 200`
- `strs[i]` 如果非空，则仅由小写英文字母组成

---

## 解题思路

### 方法一：纵向扫描

从左到右遍历所有字符串的每一列，比较相同位置上的字符是否相同。

**时间复杂度：** O(mn)，其中 m 是字符串的平均长度，n 是字符串的个数  
**空间复杂度：** O(1)

### 方法二：横向扫描

依次遍历字符串数组，用前两个字符串的公共前缀与下一个字符串比较。

**时间复杂度：** O(mn)
