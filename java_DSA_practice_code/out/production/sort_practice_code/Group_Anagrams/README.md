# 字母异位词分组 (Group Anagrams)

## 题目描述

给你一个字符串数组，请你将 **字母异位词** 组合在一起。可以按任意顺序返回结果列表。

**字母异位词**：由相同字母重新排列形成的单词（如 "eat" 和 "tea"）

---

## 示例

### 示例 1

**输入:**
```

strs = ["eat", "tea", "tan", "ate", "nat", "bat"]
```
**输出:**
```

[["bat"], ["nat", "tan"], ["ate", "eat", "tea"]]
```
**解释：**
- "bat" 无法通过其他字符串重新排列形成
- "nat" 和 "tan" 是字母异位词
- "ate"、"eat" 和 "tea" 是字母异位词

---

### 示例 2

**输入:**
```

strs = [""]
```
**输出:**
```

[[""]]
```
---

### 示例 3

**输入:**
```

strs = ["a"]
```
**输出:**
```

[["a"]]
```
---

## 提示

- `1 <= strs.length <= 10^4`
- `0 <= strs[i].length <= 100`
- `strs[i]` 仅包含小写字母
