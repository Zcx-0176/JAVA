# 旋转链表

## 题目描述

给你一个链表的头节点 `head`，旋转链表，将链表每个节点向右移动 `k` 个位置。

## 示例

### 示例 1

**输入：** head = [1,2,3,4,5], k = 2  
**输出：** [4,5,1,2,3]

**解释：**
```

原始：1 -> 2 -> 3 -> 4 -> 5
↓
向右移动 1 位：5 -> 1 -> 2 -> 3 -> 4
↓
向右移动 2 位：4 -> 5 -> 1 -> 2 -> 3
```
### 示例 2

**输入：** head = [0,1,2], k = 4  
**输出：** [2,0,1]

**解释：**
```

原始：0 -> 1 -> 2
↓
移动 1 位：2 -> 0 -> 1
↓
移动 2 位：1 -> 2 -> 0
↓
移动 3 位：0 -> 1 -> 2（回到原样）
↓
移动 4 位：2 -> 0 -> 1
```
## 提示

- 链表中节点的数目在范围 `[0, 500]` 内
- `-100 <= Node.val <= 100`
- `0 <= k <= 2 * 10^9`

## 解题思路

### 方法：闭合为环 + 断开

1. **计算链表长度**：遍历链表得到长度 `n`
2. **形成环**：将尾节点连接到头节点，形成环形链表
3. **找到新头节点**：向右移动 `k` 位相当于从后数第 `k % n` 个节点作为新头
   - 实际需要移动的步数：`k = k % n`（避免重复旋转）
   - 新头节点的位置：从原头节点向前移动 `n - k` 步
4. **断开环**：在新头节点的前一个位置断开连接

### 关键点

- **取模运算**：`k = k % n`，因为旋转 `n` 次等于没有旋转
- **找到断点**：需要找到新头节点的前一个节点来断开环

## 复杂度分析

- **时间复杂度**：O(n)，其中 n 是链表的长度。需要遍历链表两次。
- **空间复杂度**：O(1)，只使用了常数级别的额外空间。
