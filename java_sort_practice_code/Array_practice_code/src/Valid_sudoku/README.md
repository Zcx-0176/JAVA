# 有效的数独 (Valid Sudoku)

## 题目描述

请你判断一个 9 x 9 的数独是否有效。只需要 **根据以下规则** ，验证已经填入的数字是否有效即可。

- 数字 1-9 在每一行只能出现一次。
- 数字 1-9 在每一列只能出现一次。
- 数字 1-9 在每一个以粗实线分隔的 3x3 宫内只能出现一次。

**注意：**

- 一个有效的数独（部分已被填充）不一定是可解的。
- 只需要根据以上规则，验证已经填入的数字是否有效即可。
- 空白格用 `'.'` 表示。

---

## 示例

### 示例 1

**输入：**
```

board =
[["5","3",".",".","7",".",".",".","."]
,["6",".",".","1","9","5",".",".","."]
,[".","9","8",".",".",".",".","6","."]
,["8",".",".",".","6",".",".",".","3"]
,["4",".",".","8",".","3",".",".","1"]
,["7",".",".",".","2",".",".",".","6"]
,[".","6",".",".",".",".","2","8","."]
,[".",".",".","4","1","9",".",".","5"]
,[".",".",".",".","8",".",".","7","9"]]
```
**输出：** `true`

---

### 示例 2

**输入：**
```

board =
[["8","3",".",".","7",".",".",".","."]
,["6",".",".","1","9","5",".",".","."]
,[".","9","8",".",".",".",".","6","."]
,["8",".",".",".","6",".",".",".","3"]
,["4",".",".","8",".","3",".",".","1"]
,["7",".",".",".","2",".",".",".","6"]
,[".","6",".",".",".",".","2","8","."]
,[".",".",".","4","1","9",".",".","5"]
,[".",".",".",".","8",".",".","7","9"]]
```
**输出：** `false`

**解释：** 除了第一行的第一个数字从 5 改为 8 以外，空格内其他数字均与 示例 1 相同。但由于位于左上角的 3x3 宫内有两个 8 存在，因此这个数独是无效的。

---

## 约束条件

- `board.length == 9`
- `board[i].length == 9`
- `board[i][j]` 是一位数字（1-9）或者 `'.'`

---

## 解题思路

### 方法一：一次遍历哈希表 ⭐

使用三个哈希表分别记录每行、每列、每个 3x3 宫内的数字出现情况。遍历整个数独板，检查是否有重复数字。

**关键点：**
- 使用 `HashSet` 或布尔数组记录已出现的数字
- 对于位置 `(i, j)`，其所属的 3x3 宫索引可通过公式计算：`boxIndex = (i / 3) * 3 + (j / 3)`
- 一旦发现某个数字在同行、同列或同宫已存在，立即返回 `false`

**算法步骤：**
1. 创建三个二维布尔数组：
   - `row[9][9]`：记录每行中数字 1-9 的出现情况
   - `col[9][9]`：记录每列中数字 1-9 的出现情况
   - `box[9][9]`：记录每个 3x3 宫中数字 1-9 的出现情况
2. 遍历数独板的每个位置 `(i, j)`
3. 如果当前位置不是 `'.'`：
   - 计算数字对应的索引（0-8）
   - 计算所在 3x3 宫的索引
   - 检查该数字是否已在当前行、列、宫中出现过
   - 如果出现过，返回 `false`；否则标记为已出现
4. 遍历完成后返回 `true`

- **时间复杂度：** O(1)，因为数独板大小固定为 9×9
- **空间复杂度：** O(1)，使用固定大小的辅助数组

## 注意事项

1. **只需验证，不需求解**：题目只要求验证已填入数字的有效性，不要求填满整个数独
2. **空白格处理**：`.` 表示空白格，不参与验证
3. **字符类型**：输入是字符数组，数字以字符形式存储（'1'-'9'）
4. **索引转换**：字符转数字索引时需减去 `'1'` 或 `'0'`
5. **3x3 宫索引计算**：掌握公式 `boxIndex = (i / 3) * 3 + (j / 3)`
