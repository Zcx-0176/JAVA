# 合并K个有序链表

给你一个链表数组，每个链表都已经按升序排列。

请你将所有链表合并到一个升序链表中，返回合并后的链表。

## 示例

**示例 1：**

输入：`lists = [[1,4,5],[1,3,4],[2,6]]`  
输出：`[1,1,2,3,4,4,5,6]`  
解释：链表数组如下：
```

[
1->4->5,
1->3->4,
2->6
]
```
将它们合并到一个有序链表中得到：  
`1->1->2->3->4->4->5->6`

**示例 2：**

输入：`lists = []`  
输出：`[]`

**示例 3：**

输入：`lists = [[]]`  
输出：`[]`

## 提示

- k == lists.length
- 0 <= k <= 10^4
- 0 <= lists[i].length <= 500
- -10^4 <= lists[i][j] <= 10^4
- lists[i] 按升序排列
- lists[i].length 的总和不超过 10^4

## 算法说明

采用分治法（Divide and Conquer）策略：
1. 将链表数组递归地分成两半
2. 分别合并左右两部分的结果
3. 最后将两个已排序的结果合并成一个有序链表
