# 在排序数组中查找元素的第一和最后一个位置 (Search for Range)

## 题目描述

给你一个按照**非递减顺序**排列的整数数组 `nums`，和一个目标值 `target`。请你找出给定目标值在数组中的**开始位置**和**结束位置**。

如果数组中不存在目标值 `target`，返回 `[-1, -1]`。

**要求：** 你必须设计并实现时间复杂度为 `O(log n)` 的算法解决此问题。

---

## 示例

### 示例 1

**输入：** `nums = [5,7,7,8,8,10], target = 8`  
**输出：** `[3,4]`

### 示例 2

**输入：** `nums = [5,7,7,8,8,10], target = 6`  
**输出：** `[-1,-1]`

### 示例 3

**输入：** `nums = [], target = 0`  
**输出：** `[-1,-1]`

---

## 提示

- `0 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`
- `nums` 是一个**非递减数组**
- `-10^9 <= target <= 10^9`

---

## 解题思路

使用**二分查找**法：

1. **查找左边界**：使用二分查找找到第一个等于 `target` 的位置
    - 当 `nums[mid] >= target` 时，向左收缩右边界
    - 最终检查 `left` 位置是否等于 `target`

2. **查找右边界**：使用二分查找找到最后一个等于 `target` 的位置
    - 当 `nums[mid] <= target` 时，向右收缩左边界
    - 最终检查 `right` 位置是否等于 `target`

3. **返回结果**：将左右边界组成数组返回

**时间复杂度：** O(log n)  
**空间复杂度：** O(1)
