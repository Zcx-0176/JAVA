## <center><font face="楷体" size=5>zcx</font>   
### 一、使用IDEA创建第一个Java程序    
1. 创建工程new project(空工程)   
2. 创建模块new module   
3. 创建包new package(包名为com.域名倒写.技术名称)   
4. 写程序,创建类     
5. 写输出函数println快捷写法"输出内容".sout回车     
### 二、安装AI插件   
1. 在菜单-设置(setting)-插件(plugins)处搜索通义    
2. 在代码块中使用//注释后回车,可让AI直接写代码,使用tab键采纳AI所写的代码   
### 三、IDEA设置   
1. 字体设置:setting-editor-font      
### 四、快捷键    
| 快捷键 | 功能 |
| :---: | :---: |      
| ctrl+alt+L | 格式化代码 |    
| main/psvm | 快速键入相关代码 |   
| sout | 快速键入println()方法 |    
| ctrl+Y | 删除一行 |   
| ctrl+D | 复制该行到下一行 |   
| ctrl+alt+T | 快速生成if...else |    
| ctrl+shift+/ | 注释整段代码 |   
| ctrl+shift+/ | 取消注释整段代码 |   
| ctrl+/ | 注释一行代码 |     
### 五、IDEA中的其他操作    
| 方式 | 效果 |
| :---: | :---: |    
| File-Open(最好先复制要打开的工程的路径)然后粘贴上去,直接找也行,需要选择有黑色的点的文件 | 打开工程 |    
| 直接叉掉 | 关闭工程 |   
| 去磁盘把一整个工程删除 | 删除工程 |
| 在类处右键-Delete | 删除类文件 |    
| 在类处右键-Rename | 修改类名称 |   
| 在模块处右键-Rename-rename module and directory | 修改模块名称 |   
| 在磁盘中找到模块Ctrl+c,选中要导入模块的工程ctrl+v,接着File-new-module from existing sources-在磁盘中寻找你要导入的模块-一直点OK-导入后jdk可能不一致需要再选中当前工程的jdk-若jdk选中后出错重启就好 | 导入模块 |    
| 选中要删除的模块右键-remove module-使得该模块不再该工程中-接着右键delete module-删除 | 删除模块 |
